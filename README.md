# Mono

A minimalist, modular app launcher and utility suite for Apple devices, inspired by the focused aesthetic of E-Ink.

[![Website](https://img.shields.io/badge/Website-Visit-4A90E2)](https://0xleif.github.io/Mono/)

## About

Mono is a utility suite for focused, distraction-free workflows. Built in Swift with SwiftUI, Mono offers a quiet, clutter-free home screen and a curated set of tools that stay out of your way.

**Design Philosophy:**
- **Minimalist** - Monospaced UI with a pure black-and-white palette
- **Low-Stimulation** - Little to no motion, color, or haptics
- **Modular** - Every feature lives in its own independent Swift Package
- **E-Ink Inspired** - High-contrast, low-motion interface for clarity and focus

Visit [our website](https://0xleif.github.io/Mono/) to learn more about Mono's features, applets, and roadmap.
